Upload New Voice Sample

Record a 30-second clip of the person speaking naturally. Longer samples give better results. Supported formats: WAV, MP3, M4A, OGG, FLAC.

How It Works

1. Upload a 30-second audio clip of the person's voice

2. ElevenLabs creates a voice clone from the sample

3. Click "Activate" to make Reachy speak in that voice

4. Use "Preview" to hear what it sounds like before enabling

⚠️ Requires an ElevenLabs API key set in the robot's .env file (ELEVENLABS_API_KEY)
